2.	Registration (Story Protocol Layer):
o	Parent IP: The original character or universe is registered as a root IP Asset.
o	Licensing (PIL): The creator attaches a Programmable IP License.

o	Flow: Revenue from the derivative automatically flows upstream to the Parent IP and the Platform.
